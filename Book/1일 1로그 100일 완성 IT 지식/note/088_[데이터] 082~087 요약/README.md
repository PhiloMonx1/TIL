### 디지털 기술 사용과 데이터 수집
1. 디지털 기술 사용과 데이터 수집
   - 방대한 데이터 흐름 생성
   - 예상보다 큰 규모의 상세한 데이터 수집
   - 상업적 용도로 활용 (공유, 결합, 연구, 판매)
2. 무료 서비스의 대가
   - 검색, SNS, 휴대전화 앱, 온라인 저장소 등의 서비스 이용 시 데이터 제공
3. 대중의 인식 변화
   - 데이터 수집 범위에 대한 인식 증가
   - 광고 차단 프로그램 사용 증가
4. 광고 차단의 영향
   - 웹 생태계 변화 가능성
   - 대안적 비즈니스 모델 필요성
