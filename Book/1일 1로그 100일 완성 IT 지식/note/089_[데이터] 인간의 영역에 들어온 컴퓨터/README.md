## 089 [데이터] 인간의 영역에 들어온 컴퓨터

---

### 인공지능의 발전과 현황
- 발전 요인
  - 컴퓨팅 성능 향상
  - 메모리 기술 발전
  - 대량의 데이터 활용
  - 복잡한 수학 공식 적용
- 인공지능의 성공 사례
  - 게임 (체스, 바둑)
  - 음성 인식 및 합성
  - 기계 번역
  - 영상 인식 및 컴퓨터 비전
  - 자율주행 자동차
  - 추천 시스템
  - 스팸 탐지

### 주요 용어
- 인공지능 (AI)
  - 인간의 지능을 모방하는 컴퓨터 시스템
- 머신러닝 (ML)
  - AI의 부분집합
  - 알고리즘을 훈련시켜 결정을 내리게 하는 기술
  - 통계와 유사하지만 다른 접근 방식
- 딥 러닝 (DL)
  - 머신러닝의 특정 형태
  - 인간 뇌의 신경망을 모방한 계산 모델 사용
  - 컴퓨터 비전 분야에서 특히 성과가 뛰어남

### 머신러닝의 특징
- 통계와 달리 모델을 가정하지 않음
- 데이터에서 직접 관계를 찾음
- 대규모 데이터셋에 적용
- 확률적 결과 도출
