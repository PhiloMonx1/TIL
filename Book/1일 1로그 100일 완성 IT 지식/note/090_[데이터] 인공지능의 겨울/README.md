## 090 [데이터] 인공지능의 겨울

---

### 초기 인공지능 연구 (20세기 중반)
- 주요 연구 분야
  - 게임 (체커, 체스)
  - 기계 번역
  - 음성 인식 및 생성
  - 수학적/논리적 추론
  - 의사 결정
  - 학습 과정 연구
- 연구 자금
  - 주로 정부 기관 지원
- 1950-60년대 분위기
  - 매우 낙관적 전망
  - 5-10년 내 획기적 혁신 기대

### AI 겨울 (1970-80년대 초반)
연구 난관 및 자금 부족으로 인한 침체기

### 전문가 시스템 (1980-90년대)
- 작동 원리
  - 전문가가 규칙 작성
  - 프로그래머가 규칙을 코드로 변환
  - 컴퓨터가 코드 적용하여 과제 수행
- 주요 응용 분야
  - 의료 진단 ex) MYCIN
  - 고객 지원
  - 기계 시스템 유지/수리
- 한계점
  - 모든 고려사항을 포함한 규칙 집합 구성의 어려움
  - 예외 처리의 복잡성
  - 새로운 문제로의 확장성 부족
  - 지속적인 규칙 업데이트 필요성
