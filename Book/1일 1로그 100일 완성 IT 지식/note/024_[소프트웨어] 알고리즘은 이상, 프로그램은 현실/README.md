## 024 [소프트웨어] 알고리즘은 이상, 프로그램은 현실

---

### 알고리즘과 프로그램의 간극
- 알고리즘 : 추상적이고 이상적인 절차를 기술한 것이다.
- 프로그램 : 실제 컴퓨터가 과제를 완료하기 위해 수행해야 하는 모든 단계를 구체적으로 서술한 것이다.

알고리즘과 프로그램은 청사진과 건물의 관계와 비슷하다. 

### 프로그램의 현실 제약
알고리즘과 달리 실질적인 문제도 신경써야 한다.
1. 불충분한 메모리
2. 제한된 프로세서 속도
3. 잘못 입력된 데이터
4. 하드웨어 결함
5. 네트워크 연결 불량
6. 휴먼 에러
